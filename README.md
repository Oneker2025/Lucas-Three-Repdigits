# Supplementary Material

This repository contains the supplementary material for the paper

**Lucas Numbers as Concatenations of Three Repdigits**

## Contents

### paper/

Contains the complete supplementary tables.

### code/

Contains the Mathematica notebook used to reproduce all computations and tables presented in the paper.

The notebook was developed and tested with Wolfram Mathematica.

Running the notebook reproduces all computational results reported in the paper.
